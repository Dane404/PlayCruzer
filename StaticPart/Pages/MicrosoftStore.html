Microsoft Store

Publisher: Microsoft

Description:

The Microsoft Store is the store which Microsoft developed. You can use it on the new Xbox Consoles (since 2017), on the Windows Phones and on the PC.

Plattforms:

Windows: 2012 as Windows Store
Windows 10 Mobile Phone: 09.2017
Xbox One: 09.2017, former Xbox Store got replaced with the re-branded Microsoft Store (former Windows Store)
Xbox Series X: 10.11.2020 on the release date of the Xbox series X

Release:

26.10.2012 as Windows Store

History and Evolution:

The Microsoft Store started 2008 as a Web-based store. On Windows 8 the Windows Store was announced in 2011 but it was first released in 2012. After Windows 10 came out, all Microsoft Services were united to one store. In 2017, Microsoft re-branded the Windows Store and it became the Microsoft Store. The Xbox Store on the Xbox also got merged and became replaced with the Microsoft Store.

Advantages:

  • When you buy an “Xbox Play Anywhere” as a digital title, you can play it on pc and on the Xbox

Disadvantages:

  • Does not have many exclusive titles like other Console Stores (PS Store: GoW, Spiderman ps4 etc. E-Shop: Mario Games, Zelda Games etc.)
